# Claude Prompts — SKAI Команда 7

**Правило:** один промпт = один файл.

```
claude-prompts/
│
├── stitch/              ← UI-экраны для Google Stitch (копируй → вставляй)
│   ├── 01-incident-window.md       экран карточки инцидента
│   ├── 02-events-feed.md           лента событий
│   ├── 03-bv-report-video.md       отчёт БВ + клик → видео (Оздоев killer)
│   ├── 04-driver-training.md       обучение водителя после события
│   └── 05-dispatch-alert.md        диспетчерский алерт ±15 сек видео
│
├── hackathon/           ← технические промпты по волнам (Cursor / Claude Code)
│   ├── wave-00/         1 агент   — прочитать контекст
│   ├── wave-01/         5 агентов — types / constants / incidents / vehicles / infra
│   ├── wave-02/         7 агентов — компоненты (по одному файлу каждый)
│   ├── wave-03/         3 агента  — экраны (сборка)
│   ├── wave-04/         1 агент   — роутинг
│   └── wave-05/         4 агента  — таблица, экран заявок, smoke, демо
│
└── ajtbd/               ← продуктовые промпты (методология AJTBD / Замесина)
    ├── wave-01/         параллельно: b2b-сегменты + анализ интервью
    ├── wave-02/         после волны 1: RAT риски
    └── wave-03/         после волны 2: граф работ
```

## Как использовать Stitch

1. Открой https://stitch.withgoogle.com
2. Скопируй содержимое `DESIGN.md` → вставь как первый контекст
3. Скопируй нужный экран из `stitch/` → вставь как промпт
4. Генерируй → экспортируй React/Tailwind

## Как использовать Hackathon промпты

Волна = параллельный запуск. Запускай следующую волну только после завершения предыдущей.  
Каждый агент трогает **только свой файл**.
