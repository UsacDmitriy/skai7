Диспетчер вводит запрос голосом или текстом → система подтверждает понимание →
строит интерактивный дашборд. Два режима: В-1 (один водитель) / В-2 (весь парк).
Killer feature (Оздоев, ГПН): «Нажал нарушение → вылезло видео».

# Экран 5 — Отчёт «Безопасное вождение»: NL-запрос → подтверждение → интерактивный отчёт

## Что рисуем
Экран работает в трёх состояниях:

СОСТОЯНИЕ А — Пустой (до ввода): поле ввода + пресеты.
СОСТОЯНИЕ Б — Подтверждающее окно: система показывает как поняла запрос.
СОСТОЯНИЕ В — Полный отчёт: водитель + ТС + пробег + статистика + трек + график + таблица нарушений + видео.

**Главный NL-сценарий** (ключевая идея продукта):
Диспетчер пишет: «Покажи нарушения по ВА и телематике у Иванова за последние три дня»
→ Система подтверждает понимание
→ Строит полный дашборд по водителю за период

Killer feature по Оздоеву (ГПН, 800 ТС):
«Как Sky Studio: нажал превышение — вылез трек.
А тут нажал нарушение — вылезло видео. Это было бы просто здорово.»

### NLU — определение режима и параметров по тексту запроса

```ts
function parseQuery(q: string): ParsedQuery {
  const isFleet = /компани|парк|все водител|подразделен|топ|флот/i.test(q)

  // Извлечение имени водителя: ищет "Иванов", "Петров", "Сидоров" и т.д.
  const driverNames = ['Иванов', 'Петров', 'Сидоров', 'Козлов', 'Новиков']
  const foundDriver = driverNames.find(name =>
    new RegExp(name, 'i').test(q)
  )

  // Извлечение периода в днях: ищет "за X дней", "за X дня", "за неделю", "за месяц"
  let periodDays = 3
  const dayMatch = q.match(/за\s+(\d+)\s+(дня|дней|день)/i)
  if (dayMatch) periodDays = parseInt(dayMatch[1])
  if (/недел/i.test(q)) periodDays = 7
  if (/месяц/i.test(q)) periodDays = 30

  const now = new Date()
  const start = new Date(now.getTime() - periodDays * 86400000)
  const periodLabel = `${start.getDate().toString().padStart(2,'0')}.${(start.getMonth()+1).toString().padStart(2,'0')}–${now.getDate().toString().padStart(2,'0')}.${(now.getMonth()+1).toString().padStart(2,'0')}.${now.getFullYear()}`

  if (isFleet) {
    return { mode: 'fleet', periodDays, periodLabel, dataTypes: ['va','tel'], confidence: 0.94 }
  } else if (foundDriver) {
    return {
      mode: 'single_driver',
      driver: `%FULL_NAME_FOR_${foundDriver}%`,  // агент подставит полное ФИО из driver-report.json
      vehicleId: '%PLATE_FOR_DRIVER%',            // агент подставит госномер
      periodDays,
      periodLabel,
      dataTypes: ['va','tel'],
      confidence: 0.91
    }
  } else {
    return {
      mode: 'single_driver',
      driver: 'Иванов Алексей Петрович',
      vehicleId: 'А777ВВ 77',
      periodDays,
      periodLabel,
      dataTypes: ['va','tel'],
      confidence: 0.85  // ниже уверенность — не нашли имя в запросе
    }
  }
}
```
