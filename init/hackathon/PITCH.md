# Что мы делаем

**Команда 7 · Тема 5**

---

## Одной фразой

Диспетчер открывает один продукт — и видит где машины, что происходит,
видео к каждому событию и кнопки действий. Не две системы. Одна.

---

## Проблема

Олег Фомин, PepsiCo:
> «Мы прыгаем: сначала смотрим видеоаналитику, потом телематику.
> Заходим в два разных отчёта. Теряется много времени.»

За год — 39 ДТП. С видеодоказательством расследованы 5.

Руслан Оздоев, Газпромнефть (800 машин):
> «Нажал нарушение в отчёте — вылезло бы видео.
> Как в телематике: нажал превышение — вылез трек.
> А тут нажал — вылезет видео. Это было бы просто здорово.»

ГПН уже строят внутреннюю платформу SMAP LPS 2.0 именно для этого.

## Экраны

**4 основных:**

| Экран | Что делает |
|-------|-----------|
| 1. Карта | Все машины, цвет по риску, лента событий справа |
| 2. Карточка события | Видео + CAN-скорость + акселерометр синхронно — главный экран |
| 3. Диалог аналитики | Голосом или текстом → подтверждение → дашборд: водитель, трек, график скорости, ВА+телематика, клик → видео. Whisper AI — без рук |
| 4. Список событий | Таблица потока, KPI вверху, фильтр и поиск |

**+ расширенные (если успеем):**

| Экран | Что делает |
|-------|-----------|
| 5. Отчёт БВ | Телематика + видеодетекции, клик → видео (Оздоев killer feature) — тот же NL-поток, только специализированный шаблон |
| 6. Диспетчерский алерт | Событие → авто-запрос видео ±15 сек → кнопки диспетчера |
| 7. Досье рейса | Весь рейс на карте с видеоотметками |
| 8. Восстановление при РЭБ | Маршрут по видео когда GPS заглушен |

## Три сценария для жюри

**Flow 1 — ДТП без камеры водителя (Петров, В345КМ 97):**
Красный маркер → карточка → скорость 54→0, акселерометр пик →
«Создать заявку» → статус изменился.

**Flow 2 — NL-запрос по водителю (КЛЮЧЕВОЙ — Оздоев / ГПН):**
Голосом или текстом: «Покажи нарушения Иванова за три дня» или «Все нарушения по компании» →
подтверждающее окно → два режима результата:
В-1 (один водитель): трек + график + ВА+телематика + клик→видео
В-2 (fleet / парк): переключатель «👤 По водителям | 🚛 По ТС»
  — По ТС: статус камер каждого авто, водители за период, [📷 CAM-03 offline] красный
Голосовой ввод — faster-whisper large-v3 (RU/KK/EN), без рук.
Анализ текста — Groq API + LLaMA 3.3 70B (бесплатный тариф).
Языки: 🇷🇺 русский · 🇰🇿 казахский · 🇬🇧 английский — нативно.
Полностью open-source, запускается локально или через дешёвый API.

**Flow 3 — Нет видео (Сидоров, Е902СТ 150):**
Пометка «Нет видео» → карточка → объяснение почему →
«Запросить архив» → «Запрос создан».
