# Идея #2 (P0) — Интерактивный аналитический отчёт

> Приоритет: **#2 из 2**. Второе по важности для демо — после единого окна.
> Экран: 5 (Диалог аналитики)

## Суть

Диспетчер говорит или пишет запрос → система подтверждает понимание → строит дашборд.
Два режима результата: один водитель (В-1) или весь парк (В-2).
В-2 имеет переключатель «По водителям | По ТС».

## Источник потребности

**Руслан Оздоев (ГПН, 800 ТС, 5 стран):**
> «Нажал нарушение в отчёте — вылезло бы видео. Как в телематике:
>  нажал превышение — вылез трек. А тут нажал — вылезет видео. Это было бы просто здорово.»
> «Важно, что из отчёта можно перейти в видео.»
ГПН строит SMAP LPS 2.0 именно под этот сценарий — подтверждённый рыночный спрос.

**Олег Фомин (PepsiCo):**
> «Если есть нарушения — по каждому моменту времени будет видеоролик тоже дозагружен?»

## Технологии

- **STT:** faster-whisper large-v3 — локально, open-source, RU 🇷🇺 / KK 🇰🇿 / EN 🇬🇧
- **NLU:** Groq API + LLaMA 3.3 70B — бесплатный тариф, 300+ токенов/сек
- **NLU offline:** Ollama + LLaMA 3.1 8B

Пайплайн: 🎤 → faster-whisper → текст → Groq+LLaMA → ParsedQuery → подтверждение → отчёт

## Поток

1. Ввод голосом [🎤] или текстом (пресеты: 6 готовых запросов)
2. Подтверждающее окно: «Вот как я понял ваш запрос» + уверенность 96%
   Параметры: водитель · период · тип данных (ВА + Телематика)
3. Результат (NLU выбирает режим автоматически):

**Режим В-1 — конкретный водитель** («Нарушения Иванова за три дня»):
- Карточка водителя: имя, рейтинг, подразделение, регион
- Карточка ТС: госномер, модель, пробег, рейсов, время
- KPI: всего / ВА / телематика / грубых
- Таблица нарушений ВА+телематика — клик на строку → видео справа (killer feature)
- По умолчанию справа: трек маршрута + график скорости

**Режим В-2 — парк** («Все нарушения по компании за три дня»):
Переключатель «👤 По водителям | 🚛 По ТС»

«По водителям» — служба безопасности / HR:
- Рейтинг водителей, Score, badges ВА/тел/грубых
- Мини-дашборд → [Открыть полный отчёт] → В-1

«По ТС» — логисты / технические службы:
- ТС = актив компании, 1 ТС может иметь несколько водителей за период
- Статус камер: [📷 3/3 работают] зел. / [📷 2/3] жёлтый / [📷 CAM-03 offline] красный
- Водители за период: Иванов А.П. [основной] 4 рейса + Козлов И.А. 1 рейс
- Мини-дашборд ТС с историей и камерами
