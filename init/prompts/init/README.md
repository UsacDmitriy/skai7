# Prompts/

> Готовые промпты для использования с Cursor / Continue / Claude / любым LLM на хакатоне.

## Когда какой промпт

| Когда | Промпт | Кто использует |
|---|---|---|
| Утром 21 мая после объявления темы | [`discovery-prompt.md`](./discovery-prompt.md) | Вся команда вместе |
| Каждые 30-60 минут во время работы | [`orchestration-prompt.md`](./orchestration-prompt.md) | Pilot |
| Инициализация репозитория и инфраструктуры | [`hackathon-init/`](./hackathon-init/) | Любой из команды (параллельный старт) |

## Discovery Prompt

Структурированный дискавери с 12 вопросами. LLM играет роль системного аналитика, задаёт вопросы по очереди, после всех вопросов выдаёт черновик AGENTS.md.

Используется **один раз в день**, утром, после объявления темы хакатона. Занимает 10-15 минут.

## Orchestration Prompt

Координация работы нескольких AI-агентов. LLM играет роль главного архитектора, проверяет согласованность кусков от разных агентов, выдаёт action items на следующий час.

Используется **5-6 раз в день**, каждые 30-60 минут. Занимает 5-10 минут на каждый запуск.

## Принципы написания промптов в этой папке

Оба промпта построены по одной модели:

1. **Роль** — LLM получает чёткую персону и контекст
2. **Правила поведения** — что LLM делает и не делает
3. **Контекст хакатона** — общая информация
4. **Слоты для команды** — куда вставить специфику дня
5. **Структурированный output** — формат ответа
6. **Поведение после output** — что делать дальше

Это паттерн для любого хакатонного промпта. Если будете создавать свои — следуйте этой структуре.

## Что не вошло в эту папку

В каждой из 21 идей в `ideas/` лежит свой `prompt-codex.md` — это **стартовый промпт для запуска Cursor Composer / Codex** на конкретной задаче. Он специфичен для idea и заполнен.

В `templates/prompt-codex.md.template` — универсальный шаблон стартового промпта, заполняется утром 21 мая под реальную тему.

В `playbook/09-stitch-cookbook.md` — техники и шаблоны промптов для Stitch.

Эти три места вместе с `prompts/` покрывают весь жизненный цикл промптов на хакатоне:

```
Discovery (prompts/discovery-prompt.md)
  ↓
Заполнение шаблона (templates/AGENTS.md.template)
  ↓
Стартовый промпт для агента (templates/prompt-codex.md.template)
  ↓
UI-бриф (templates/prompt-stitch.md.template + playbook/09-stitch-cookbook.md)
  ↓
Orchestration каждые 30-60 минут (prompts/orchestration-prompt.md)
  ↓
Финальная полировка
```
